Reference HTML Cheatsheet
Tag Name Description

<!DOCTYPE>

Doc Type

Defines the document type

<a>

Anchor

Defines a hyperlink

<address>

Address

Defines contact information for the author/owner of a document

<article>

Article

Defines an article (independent, self-contained content)

<aside>

Aside

Defines content aside from the page content (indirectly related
  to the surrounding content, but not necessary to understand it)

<audio>

Audio

Defines embedded sound content

<br>

Line

Break Defines a single line break

<button>

Button

Defines a clickable button

<caption>

Caption

Defines a table caption

<div>

Division

Defines a section in a document

<em>

Emphasized

Defines emphasized text

<embed>

Embed

Defines a container for an external application

<figcaption>

Figure Caption

Defines a caption for a <figure> element.

<figure>

Figure

Specifies self-contained content

<footer>

Fotter

Defines a footer for a document or section

<h1> to <h6>

Headings

Defines HTML heading

<header>

Header

Defines a header for a document or section

<img>

Image

Defines an image

<input>

Input

Defines an input control

<link>

Link

Defines the relationship between a document and an external
  resource (most used to link to style sheets)

<nav>

Navigation

Defines navigation link

<pre>

Preformat

Defines preformatted text

<script>

Script

Defines a client-side script

<section>

Section

Defines a section in a document

<span>

Span

Defines a section in a document

<strong>

Strong

Defines important text

<table>

Table

Defines a table

<tbody>

Table Body

Groups the body content in a table

<td>

Table Data

Defines a cell in a table

<tfoot>

Table Foot

Groups the footer content in a table

<th>

Table Header

Defines a header cell in a table

<thead>

Table Head

Groups the header content in a table

<title>

Title

Defines a title for the document

<tr>

Table Row

Defines a row in a table

<video>

Video

Defines embedded video content
